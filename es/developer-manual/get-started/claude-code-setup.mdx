**Claude Code** es la herramienta de línea de comandos de Anthropic para interactuar con los modelos Claude. Te permite usar las capacidades de Claude directamente desde tu terminal para tareas de desarrollo y análisis.

Para instrucciones de instalación, consulta la [documentación oficial de Claude Code](https://claude.com/product/claude-code).

<Note>
  Esta guía fue probada el 16 de abril de 2026. Si no funciona, consulta otras alternativas en la [guía de configuración de Opencode](./opencode-setup).
</Note>

## Pasos de Configuración

<Steps>
  <Step title="Crear un token de Gateway">
    Antes de configurar Claude Code, necesitas un **token de Gateway** de Zylon.

    Consulta [Gestión de Tokens](./token-management) para obtener instrucciones detalladas sobre cómo crear tokens de Gateway.
  </Step>

  <Step title="Establecer variables de entorno">
    Agrega lo siguiente a tu archivo de configuración de shell (`~/.zshrc` o `~/.bashrc`):

    ```bash
    echo 'export ZYLON_BASE_URL="https://your-host/api/gpt"' >> ~/.zshrc
    echo 'export ZYLON_API_TOKEN="your-gateway-token"' >> ~/.zshrc
    echo 'export ZYLON_MODEL="default"' >> ~/.zshrc
    source ~/.zshrc
    ```

    Reemplaza `your-host` con la URL de tu instancia de Zylon y `your-gateway-token` con tu token de Gateway real. El modelo predeterminado es `default`.

    <Warning>
      Nunca hagas commit de variables de entorno que contengan tokens al control de versiones. Agrégalas a tu `.gitignore`.
    </Warning>

    Confirma que las variables se establecieron correctamente:

    ```bash
    echo $ZYLON_BASE_URL
    echo $ZYLON_API_TOKEN
    echo $ZYLON_MODEL
    ```
  </Step>

  <Step title="Configurar Claude Code">
    Agrega lo siguiente a tu archivo de configuración de shell (`~/.zshrc` o `~/.bashrc`):

    ```bash
    echo 'export ANTHROPIC_BASE_URL="$ZYLON_BASE_URL"' >> ~/.zshrc
    echo 'export ANTHROPIC_AUTH_TOKEN="$ZYLON_API_TOKEN"' >> ~/.zshrc
    echo 'export ANTHROPIC_MODEL="$ZYLON_MODEL"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Step>

  <Step title="Ejecutar Claude Code">
    Prueba tu configuración ejecutando:

    ```bash
    claude-code
    ```

    Si está configurado correctamente, Claude Code se conectará a tu instancia de Zylon y estará listo para ayudarte.
  </Step>
</Steps>

## Limitaciones

- Las imágenes no son compatibles

## Solución de Problemas

### Errores de conexión

- Verifica que `ZYLON_BASE_URL` sea correcta y accesible
- Asegúrate de que tu token de Gateway sea válido y no haya expirado
- Comprueba la conectividad de red con tu instancia de Zylon

### Fallos de autenticación

- Confirma que el token se copió correctamente sin espacios en blanco adicionales
- Regenera el token si pudo haber sido comprometido
- Verifica que el token tenga los permisos apropiados
