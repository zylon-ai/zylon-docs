---
title: "Registro de Auditoría"
---

Almacena eventos y trazas del Registro de Auditoría.

### Configuración

| Clave      | Por Defecto | Valores          | Descripción                      |
| -------- | ------- | --------------- | -------------------------------- |
| `enabled`| `true`  | `true \| false` | Almacena eventos y trazas del Registro de Auditoría  |

### Ejemplo

```yaml
features:
    audit:
        enabled: true
```
