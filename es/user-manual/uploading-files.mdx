title: Subir archivos

Cada proyecto tiene su **propia base de conocimiento**. Puedes poblarla mediante:

- **Subir archivos desde tu dispositivo.** Arrastra y suelta o selecciona archivos. Las subidas locales son **estáticas**—vuelve a subir si la fuente cambia.
- **Importar mediante un conector (p. ej., SharePoint).** Puedes seleccionar **archivos o carpetas completas** a las que tengas acceso. Se muestra el progreso de ingesta.
