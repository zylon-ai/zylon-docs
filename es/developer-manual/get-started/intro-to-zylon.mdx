---
title: "Intro a Zylon"
description: "Plataforma de IA auto‑hosteada con endpoints de Messages compatibles con Anthropic y una Workspace API."
---

Zylon es una plataforma de IA auto‑hosteada que se ejecuta en tu infraestructura. Ofrece endpoints de Messages compatibles con Anthropic para la interacción con el modelo y una Workspace API para la gestión de organizaciones y proyectos. En esta documentación usarás **dos APIs**:

- **API de ZylonGPT**: la capa de IA para respuestas del modelo, herramientas y chat con contenido.
- **Workspace API**: el plano de control para organizaciones, usuarios, proyectos e integraciones.

Antes de comenzar, genera un token de API en [Token Management](/es/api-reference/token-management).
