---
title: "MCP Tool Connector"
---

<Note>**Feature in Alpha**</Note>

Show MCP connector tool in chats.

### Configuration

| Key      | Default | Values          | Description                          |
| -------- | ------- | --------------- | ------------------------------------ |
| `enabled`| `false` | `true \| false` | Show MCP connector tool in chats     |

### Example

```yaml
features:
    mcp:
        enabled: true
```
