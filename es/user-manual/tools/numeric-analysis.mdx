---
title: Análisis numérico
description: Analiza un CSV de tu base de conocimiento con consultas en lenguaje natural.
---

Usa **Análisis numérico** para analizar un **único CSV** almacenado en la base de conocimiento de tu proyecto.

**Cómo funciona**
- Seleccionas el archivo CSV a analizar.
- Activas la herramienta **Análisis numérico**.
- Tu prompt en lenguaje natural se traduce a una **consulta SQL** y se ejecuta contra el CSV **como si fuera una base de datos**.
- Los resultados se renderizan como **tablas o gráficos**.

**Consejos**
- El CSV debe estar **bien formado** (una sola fila de encabezado; sin fórmulas ni múltiples hojas).
- Pide **tablas** o **gráficos** específicos (barras, líneas, mapa de calor, etc.).
- Sé lo más explícito posible respecto a los **nombres de columnas a usar**. El agente no conoce tu estructura de datos.
- Mantén las consultas basadas en los datos presentes; los **factores externos** no se infieren a menos que se proporcionen como documentos o mediante una URL obtenida en el mismo chat.

> Puedes usar **Pensamiento extendido + Análisis numérico** juntos para análisis de varios pasos.
