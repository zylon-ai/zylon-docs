Provides direct access to the platform's AI layer, encompassing all core functional building blocks: LLM calls, data ingestion, embeddings, semantic search, RAG (Retrieval-Augmented Generation), tools, agents, MCP integrations, and more.

This API enables you to build custom enterprise agents and create sophisticated AI-powered workflows tailored to your specific business requirements.
