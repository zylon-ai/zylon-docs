Usa estos endpoints para crear cuentas con credenciales, autenticar usuarios y cerrar sesión. Para los redireccionamientos SSO (Google/Microsoft) y OAuth de integraciones, consulta [OpenID & OAuth](/es/developer-manual/build-with-zylon/workspace-api/authentication/openid).

## Solicitud básica y respuesta

Crea una cuenta con credenciales usando `POST /v1/auth/register`.

```bash
curl -X POST "https://{BASE_URL}/v1/auth/register" \
  -H "Authorization: Bearer {API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "sasha@auroralabs.com",
    "password": "S3cure!Passw0rd",
    "roles": ["AppAdmin"]
  }'
```

## Iniciar sesión con credenciales

```bash
curl -X POST "https://{BASE_URL}/v1/auth/login" \
  -H "Authorization: Bearer {API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "sasha@auroralabs.com",
    "password": "S3cure!Passw0rd"
  }'
```

## Cerrar sesión

```bash
curl -X POST "https://{BASE_URL}/v1/auth/logout" \
  -H "Authorization: Bearer {API_TOKEN}"
```

<Accordion title="Ejemplo de respuesta">
```json
"ok"
```
</Accordion>

## Errores y casos límite

- **401/403**: token inválido o ausente.
- **409**: la cuenta ya existe (register).
- **429**: demasiados intentos de login.
