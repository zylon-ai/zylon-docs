## Cambiar contraseña
1. Ve a tu perfil.
![](/images/user-manual/profile_password_recovery.png)
2. Haz clic en **cambiar contraseña** y configura tu nueva contraseña.
![](/images/user-manual/profile_change_password.png)


## Restablecer contraseña <a id="reset-password" />
Solo el administrador puede restablecer la contraseña de un usuario. Para hacerlo, el administrador necesita usar la CLI de Zylon para realizar la acción.
Si eres administrador, [aprende cómo hacerlo.](/es/operator-manual/configuration/cli#update-password)
