## Change password
1. Go to your profile.
![](/images/user-manual/profile_password_recovery.png)
2. Click on **change password** and setup your new password.
![](/images/user-manual/profile_change_password.png)


## Reset password <a id="reset-password" />
Only administrator can reset a user password. To do so the administrator needs to use Zylon CLI to perform the action.
If you are an administrator, [learn how to do it.](/en/operator-manual/configuration/cli#update-password)
