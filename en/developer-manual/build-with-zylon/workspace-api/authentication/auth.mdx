title: "Auth"
description: "Create accounts, log in, and end sessions for Workspace API access."
---

Use these endpoints to create credential-based accounts, exchange credentials for a session, and log out. For SSO login redirects (Google/Microsoft) and integration OAuth, see [OpenID & OAuth](/en/developer-manual/build-with-zylon/workspace-api/authentication/openid).

## Basic request and response

Create a credentials-based account with `POST /v1/auth/register`.

```bash
curl -X POST "https://{BASE_URL}/v1/auth/register" \
  -H "Authorization: Bearer {API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "sasha@auroralabs.com",
    "password": "S3cure!Passw0rd",
    "roles": ["AppAdmin"]
  }'
```

## Log in with credentials

```bash
curl -X POST "https://{BASE_URL}/v1/auth/login" \
  -H "Authorization: Bearer {API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "sasha@auroralabs.com",
    "password": "S3cure!Passw0rd"
  }'
```

## Log out

```bash
curl -X POST "https://{BASE_URL}/v1/auth/logout" \
  -H "Authorization: Bearer {API_TOKEN}"
```

<Accordion title="Example response">
```json
"ok"
```
</Accordion>

## Errors and edge cases

- **401/403**: missing or invalid API token.
- **409**: account already exists (register).
- **429**: too many login attempts.
