Zylon supports three installation modes to accommodate different network environments. Choose the one that best fits your security and connectivity requirements.

| Feature | Online | Semi-Airgap | Airgap |
|---------|--------|-------------|--------|
| Internet Access | Full | Limited | None on target |
| Setup Complexity | Simple | Moderate | Complex |
| Security Level | Standard | High | Maximum |
| Update Process | Simple | Required manual work | Pseudo-manual |
| Proxy Support | ✓ | ✓ | ✗ |

---

## Prerequisites

All installation modes require:

- **Hardware**: Clean Linux machine (Ubuntu 22/24) meeting [hardware requirements](/en/operator-manual/hardware-requirements)
- **Access**: Root and SSH access to the machine
- **Domain**: A domain name for Zylon (e.g., zylon.company.com)
- **SSL**: Valid SSL certificate (if not using Let's Encrypt)
- **Reboot**: Ability to reboot the machine when needed
