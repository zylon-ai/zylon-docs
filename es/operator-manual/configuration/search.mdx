---
title: "Búsqueda"
---

Para obtener una mejor experiencia durante la búsqueda, se recomienda habilitar qué idiomas planeas usar en Zylon para contenido (Chats, documentos, Flujos de Agente), el idioma por defecto es `english` pero se pueden habilitar múltiples idiomas.

```yaml
languages:
  keywordSearch: ['en', 'es', 'de']
```

</Tip>
