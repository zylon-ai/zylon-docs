---
title: "Instrucciones personalizadas de proyecto"
---

Los usuarios pueden personalizar las instrucciones del proyecto desde el dialogo de configuracion del proyecto, como se explica en [Añadir instrucciones personalizadas](/es/user-manual/adding-custom-instructions).

Estas instrucciones se aplican a nivel de proyecto e influyen en como responde Zylon en los chats.

### Configuración

Use la siguiente clave para controlar la longitud maxima de las instrucciones:

```yaml
features:
  maxInstructionsLength: 4000
```

### Detalles de configuración

| Clave | Valor predeterminado | Valores | Descripción |
| --- | --- | --- | --- |
| `maxInstructionsLength` | `4000` | Entero positivo | Longitud maxima de instrucciones aceptada tanto por la UI de Workspace como por la API |
