## Resumen
Developer Tools es el portal para desarrolladores que utilizan Zylon. Actualmente permite crear y gestionar tokens de API.

Esta sección está disponible para usuarios con permisos **Developer**.

## Crear token de API
Para crear un token de API:

1. Haz clic en **Create API Token**.
2. Introduce un nombre.
3. Genera el token.
4. Copia el hash del token proporcionado.

<Warning>
El hash del token solo es visible una vez. Al cerrar el modal, no se puede recuperar. Guarda el token de forma segura.
</Warning>
