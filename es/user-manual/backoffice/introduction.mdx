El Backoffice de Zylon es el entorno de administración centralizado para gestionar la plataforma Zylon. Opera fuera de los Workspaces y se utiliza para gestionar:

- Cuentas
- Roles y permisos
- Gateways
- Tokens de API
- Estado de los servicios de la plataforma

Está pensado para usuarios con:

- Permisos **Developer**
- Permisos **Operator**

El acceso requiere una cuenta válida con el rol correspondiente. El Backoffice es independiente del entorno de Workspace.
