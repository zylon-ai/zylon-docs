---
title: Resumir archivo
description: Produce un resumen exhaustivo de documentos largos o carpetas.
---

Elige uno o más archivos/carpetas de la KB del proyecto y solicita un **resumen**. El flow lee el **contenido completo**, tomando el tiempo necesario para la exhaustividad.

- Proporciona orientación (p. ej., "conviértelo en una guía de incorporación", "nivel de detalle: moderado").
- Edita el resultado y usa **IA en línea** para refinar estructura/formato.
