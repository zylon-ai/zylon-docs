---
title: Roles y permisos
description: El acceso a la base de conocimiento está controlado por los roles del proyecto.
icon: unlock-keyhole
---

Los permisos de la base de conocimiento se rigen por **roles del proyecto**:

- **Visualizador** — puede iniciar chats y agent flows que *usan* la Base de conocimiento; **no puede** modificar la Base de conocimiento.
- **Editor** — puede **añadir/eliminar** contenido y estructura de la Base de conocimiento.
- **Administrador / Propietario** — control completo; también puede **transferir la propiedad**.
