---
title: Cuándo usar el asistente de chat
description: Haz preguntas directas o úsalo como compañero de pensamiento para tareas abiertas.
icon: comment
---

Usa el chat para dos patrones:

1. **Preguntas y respuestas directas sobre tu base de conocimiento** — cuando sabes que la respuesta debería estar en los documentos de tu proyecto. Las respuestas incluyen **citas en línea** a pasajes exactos de la fuente.
2. **Trabajo abierto como "compañero de pensamiento"** — habilita **Pensamiento extendido** para razonamiento más profundo (p. ej., redactar respuestas, planificar reuniones o tareas de varios pasos).
