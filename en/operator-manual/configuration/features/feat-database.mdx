---
title: "Database Tool Connector"
---

<Note>**Feature in Alpha**</Note>

Show database connector tool in chats.

### Configuration

| Key      | Default | Values          | Description                               |
| -------- | ------- | --------------- | ----------------------------------------- |
| `enabled`| `false` | `true \| false` | Show database connector tool in chats     |

### Example
```yaml
features:
    database:
        enabled: false
```
