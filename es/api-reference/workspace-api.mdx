---
title: "API de Workspace"
---

Habilita acceso programático a la funcionalidad de Workspace
para gestión exhaustiva de organizaciones, proyectos y usuarios.
Esto incluye capacidades de ingesta de datos y controles de autorización, proporcionando supervisión administrativa completa de tu plataforma de IA.

Ideal para integrar fuentes de datos personalizadas y automatizar
procesos internos, eliminando flujos de trabajo manuales y optimizando operaciones en toda tu organización.

Necesitarás un token para acceder a la API,
por favor consulta la documentación de [Gestión de tokens](./token-management) para más detalles.
