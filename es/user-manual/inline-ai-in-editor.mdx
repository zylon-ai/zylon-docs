---
title: IA en línea en el editor
description: Usa IA en selecciones para mejorar o generar contenido, opcionalmente basado en tu base de conocimiento.
icon: pen-to-square
---

Selecciona texto en un documento de flow o escribe '/' para invocar IA. Úsala para **mejorar formato**, **reescribir** o **generar** contenido. Puedes elegir si la IA debe **usar la base de conocimiento del proyecto** o responder de forma genérica.

- Ejemplo: "Mejora el formato" (no uses la Base de conocimiento).
- Ejemplo: "Inserta más detalles sobre nuestros requisitos de aplicación Android" (usa la Base de conocimiento para recuperar los documentos de atención al cliente e insertar los hechos).
