title: "Integración Claromentis"
description: "Sincroniza contenido de Claromentis al workspace."
---

Las integraciones de Claromentis importan contenido de la base de conocimiento. **Habilita Claromentis a nivel de instancia primero**—consulta la guía del operador: [Configuración de Claromentis](/es/operator-manual/knowledge-base/claromentis).

## Solicitud básica y respuesta

Crea una integración de Claromentis con `POST /v1/app/integration/claromentis`.

```bash
curl -X POST "https://{base_url}/v1/app/integration/claromentis" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "svc_claro",
    "password": "claro_token_123"
  }'
```

Ejemplo de respuesta (UserIntegrationDTO):

```json
{
  "id": "clar_3c2d1b4a5f6e7d8c",
  "user_id": "user_4b7c2a1d9e5f3c8b",
  "org_id": "org_2f3a9d1c7b5e4a8f",
  "type": "Claromentis",
  "created_at": "2026-02-08T14:05:00Z",
  "updated_at": "2026-02-08T14:05:00Z"
}
```

## Actualizar credenciales

```bash
curl -X POST "https://{base_url}/v1/app/integration/claromentis/clar_3c2d1b4a5f6e7d8c" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "svc_claro",
    "password": "claro_token_456"
  }'
```

Ejemplo de respuesta:

```json
{
  "id": "clar_3c2d1b4a5f6e7d8c",
  "user_id": "user_4b7c2a1d9e5f3c8b",
  "org_id": "org_2f3a9d1c7b5e4a8f",
  "type": "Claromentis",
  "created_at": "2026-02-08T14:05:00Z",
  "updated_at": "2026-02-08T15:10:00Z"
}
```

## Obtener configuración

```bash
curl "https://{base_url}/v1/app/integration/claromentis/clar_3c2d1b4a5f6e7d8c/config" \
  -H "Authorization: Bearer {api_token}"
```

Ejemplo de respuesta (esquema no definido en OpenAPI):

```json
{
  "base_url": "https://claromentis.auroralabs.com",
  "site": "Knowledge Base"
}
```

## Iniciar sincronización

```bash
curl -X POST "https://{base_url}/v1/app/integration/claromentis/clar_3c2d1b4a5f6e7d8c/ingest" \
  -H "Authorization: Bearer {api_token}"
```

Ejemplo de respuesta (SyncEvent):

```json
{
  "type": "sync_started",
  "current_artifact_count": 6,
  "integration_item_count": 24
}
```

## Errores y casos límite

- **403**: conector no habilitado o sin permisos.
- **404**: integración no encontrada.
