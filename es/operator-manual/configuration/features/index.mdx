---
title: "Todas las Características"
---

Zylon admite activación/desactivación de características, permitiéndote habilitar o deshabilitar ciertas características bajo demanda. Esto se gestiona a través del campo `features` en tu archivo de configuración.

Cada característica puede definir su propio conjunto de claves para controlar el comportamiento.

```
features:
  audit:
    enabled: true
  database:
    enabled: true
  mcp:
    enabled: true
  webExtract:
    enabled: false
  webSearch:
    enabled: false
  multimodal-image:
    enabled: false
```
