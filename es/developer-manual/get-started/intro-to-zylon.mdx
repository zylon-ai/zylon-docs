ZylonGPT te ofrece una sola superficie de API para crear experiencias de chat que pueden **usar herramientas** y **trabajar con contenido ingerido**.

Este manual de desarrollador se organiza alrededor de estos bloques principales:

- **Mensajes**: envía el historial de conversación y recibe el siguiente mensaje del asistente.
- **Artefactos**: ingiere texto/archivos/URLs en colecciones para reutilizarlos más tarde.
- **Herramientas**: habilita capacidades integradas como búsqueda semántica, análisis tabular y web search.
