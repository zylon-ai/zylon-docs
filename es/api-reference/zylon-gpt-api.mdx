title: "API de ZylonGPT"
---

Proporciona acceso directo a la capa de IA de la plataforma, abarcando todos los bloques de construcción funcionales principales: llamadas LLM, ingesta de datos, embeddings, búsqueda semántica, RAG (Generación aumentada por recuperación), herramientas, agentes, integraciones MCP y más.

Esta API te permite construir agentes empresariales personalizados y crear flujos de trabajo sofisticados impulsados por IA adaptados a tus requisitos empresariales específicos.
