---
title: "Audit Log"
---

Store Audit Log events & traces.

### Configuration

| Key      | Default | Values          | Description                      |
| -------- | ------- | --------------- | -------------------------------- |
| `enabled`| `true`  | `true \| false` | Store Audit Log events & traces  |

### Example

```yaml
features:
    audit:
        enabled: true
```
