description: "Pick the right API: ZylonGPT for model interactions, Workspace API for organizations, users, and projects."
---

Zylon gives you two API surfaces. Use **ZylonGPT** when you want model responses, tool calls, and artifact-aware chat. Use the **Workspace API** when you need to manage organizations, users, projects, threads, and integrations.
