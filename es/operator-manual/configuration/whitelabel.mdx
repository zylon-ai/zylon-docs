### Whitelabel del Agente de Zylon

Zylon tiene soporte de whitelabel, por lo que ahora puedes personalizar el nombre del asistente de IA para que coincida con la marca de tu empresa.

Por defecto, el nombre del asistente está establecido como **Zylon**. Para cambiarlo, simplemente actualiza el campo `assistantName` en tu archivo de configuración con tu nombre preferido.

**Pautas de Nomenclatura**

- Mantén el nombre de tu agente bajo 20 caracteres.
- Evita usar mayúsculas y minúsculas mixtas (ej., "MyAIAssistant").
- No incluyas símbolos inusuales, caracteres especiales o puntuación excesiva (por ejemplo, "\$uper_Bot\!").

```
assistantName: "Daisy"
```
