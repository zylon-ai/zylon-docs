---
title: Vincular bases de conocimiento
description: Reutiliza contenido de otros proyectos sin volver a ingerirlo.
icon: link
---

Para **reutilizar** contenido de la Base de conocimiento de otro proyecto **sin duplicarlo**:

1. En **Subir archivos**, elige **Vincular desde otro proyecto**.
2. Selecciona el **proyecto fuente**, luego vincula su **Base de conocimiento completa** o **carpetas** específicas.
3. El contenido vinculado está **disponible inmediatamente** (sin re‑ingesta).

**Las actualizaciones se propagan** desde el proyecto original automáticamente—las carpetas/archivos vinculados se mantienen sincronizados con su fuente.

> **Consejo:** Usa carpetas para limitar chats. Cuando inicies un chat puedes dirigirlo a la **Base de conocimiento completa** o a una **carpeta específica** para mejorar la precisión.
