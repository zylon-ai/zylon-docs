Los endpoints de analíticas resumen cómo se usa el espacio de trabajo en proyectos y usuarios.

## Solicitud básica y respuesta

Empieza con los totales desde `GET /v1/app/organization/analytics/totals`.

```bash
curl "https://{BASE_URL}/v1/app/organization/analytics/totals" \
  -H "Authorization: Bearer {API_TOKEN}"
```

<Accordion title="Ejemplo de respuesta">
```json
{
  "projects_total": 12,
  "threads_total": 184,
  "interactions_total": 1420,
  "agent_flows_total": 36,
  "uploaded_artifacts_total": 87
}
```
</Accordion>

## Ranking de usuarios activos

```bash
curl "https://{BASE_URL}/v1/app/organization/analytics/ranking/active-users?start_date=1738291200&end_date=1738972800&limit=5" \
  -H "Authorization: Bearer {API_TOKEN}"
```

## Serie de tiempo

```bash
curl "https://{BASE_URL}/v1/app/organization/analytics/timeseries?start_date=1738291200&end_date=1738972800&type=interactions" \
  -H "Authorization: Bearer {API_TOKEN}"
```

<Accordion title="Ejemplo de respuesta">
```json
[
  { "date": "2026-02-02", "value": 18 },
  { "date": "2026-02-03", "value": 25 },
  { "date": "2026-02-04", "value": 21 }
]
```
</Accordion>

## Errores y casos límite

- **400**: rango de fechas o tipo inválido.
- **403**: permisos insuficientes.
