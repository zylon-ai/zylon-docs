---
title: "Conector de Herramienta de Base de Datos"
---

<Note>**Característica en Alpha**</Note>

Muestra la herramienta conectora de base de datos en los chats.

### Configuración

| Clave      | Por Defecto | Valores          | Descripción                               |
| -------- | ------- | --------------- | ----------------------------------------- |
| `enabled`| `false` | `true \| false` | Muestra la herramienta conectora de base de datos en los chats     |

### Ejemplo
```yaml
features:
    database:
        enabled: false
```
