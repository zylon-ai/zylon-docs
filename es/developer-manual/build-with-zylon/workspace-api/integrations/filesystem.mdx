---
title: "Integración Filesystem"
description: "Sincroniza archivos desde un filesystem compartido al workspace."
---

La integración de filesystem conecta un directorio compartido con el workspace. **Habilita el conector primero** en la guía del operador: [Configuración de Filesystem](/es/operator-manual/knowledge-base/file-system).

## Solicitud básica y respuesta

Crea una integración de filesystem con `POST /v1/app/integration/filesystem`.

```bash
curl -X POST "https://{base_url}/v1/app/integration/filesystem" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "svc_filesync",
    "password": "fs_token_123"
  }'
```

Ejemplo de respuesta (UserIntegrationDTO):

```json
{
  "id": "fs_9f8e7d6c5b4a3c2d",
  "user_id": "user_4b7c2a1d9e5f3c8b",
  "org_id": "org_2f3a9d1c7b5e4a8f",
  "type": "FileSystem",
  "created_at": "2026-02-08T14:10:00Z",
  "updated_at": "2026-02-08T14:10:00Z"
}
```

## Actualizar credenciales

```bash
curl -X POST "https://{base_url}/v1/app/integration/filesystem/fs_9f8e7d6c5b4a3c2d" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "svc_filesync",
    "password": "fs_token_456"
  }'
```

Ejemplo de respuesta:

```json
{
  "id": "fs_9f8e7d6c5b4a3c2d",
  "user_id": "user_4b7c2a1d9e5f3c8b",
  "org_id": "org_2f3a9d1c7b5e4a8f",
  "type": "FileSystem",
  "created_at": "2026-02-08T14:10:00Z",
  "updated_at": "2026-02-08T15:02:00Z"
}
```

## Obtener configuración

```bash
curl "https://{base_url}/v1/app/integration/filesystem/fs_9f8e7d6c5b4a3c2d/config" \
  -H "Authorization: Bearer {api_token}"
```

Ejemplo de respuesta (esquema no definido en OpenAPI):

```json
{
  "path": "/mnt/shared/support"
}
```

## Validar credenciales

```bash
curl "https://{base_url}/v1/app/integration/filesystem/validate?username=svc_filesync&password=fs_token_456" \
  -H "Authorization: Bearer {api_token}"
```

Ejemplo de respuesta:

```json
true
```

## Listar archivos

```bash
curl "https://{base_url}/v1/app/integration/filesystem/files?integrationId=fs_9f8e7d6c5b4a3c2d" \
  -H "Authorization: Bearer {api_token}"
```

Ejemplo de respuesta:

```json
[
  {
    "path": "/mnt/shared/support/weekly-brief.pdf",
    "name": "weekly-brief.pdf",
    "size": 287104,
    "updated_at": "2026-02-07T19:22:10Z"
  }
]
```

## Iniciar sincronización

```bash
curl -X POST "https://{base_url}/v1/app/integration/filesystem/fs_9f8e7d6c5b4a3c2d/ingest" \
  -H "Authorization: Bearer {api_token}"
```

Ejemplo de respuesta (SyncEvent):

```json
{
  "type": "sync_started",
  "current_artifact_count": 12,
  "integration_item_count": 48
}
```

## Errores y casos límite

- **403**: conector no habilitado o sin permisos.
- **400**: credenciales inválidas.
- **404**: integración no encontrada.
