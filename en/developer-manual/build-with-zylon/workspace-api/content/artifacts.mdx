---
title: "Artifacts"
description: "Ingest, update, and retrieve project artifacts in the Workspace API."
---

Artifacts are project-scoped content objects (text, files, or links) that power retrieval and analysis. Use these endpoints to upload or sync content and then read, update, or delete it.

## Basic request and response

Create a project artifact with `POST /v1/app/project/{projectId}/artifact`.

<Tabs>
  <Tab title="Text">
```bash
curl -X POST "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "InlineText",
    "source": "User",
    "name": "Q1 Support Summary",
    "description": "Weekly summary notes.",
    "raw_content": "Ticket volume rose 12% week-over-week. Billing issues drove 38% of cases."
  }'
```
  </Tab>
  <Tab title="File">
```bash
curl -X POST "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "Document",
    "source": "User",
    "name": "support-brief-q1.pdf",
    "description": "Q1 support brief (PDF).",
    "raw_content": "JVBERi0xLjQKJ..."
  }'
```
  </Tab>
  <Tab title="URI">
```bash
curl -X POST "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "Link",
    "source": "User",
    "name": "Product FAQ",
    "external_url": "https://docs.auroralabs.com/faq"
  }'
```
  </Tab>
</Tabs>

Example response (ArtifactDTO):

```json
{
  "id": "artifact_2b4d6f8a1c3e5a7b",
  "project_id": "proj_7a5c3e1b9d2f4a6c",
  "name": "Q1 Support Summary",
  "description": "Weekly summary notes.",
  "type": "InlineText",
  "source": "User",
  "state": "Ready",
  "ingest_status": "Processing",
  "has_content": true,
  "raw_content": "Ticket volume rose 12% week-over-week. Billing issues drove 38% of cases.",
  "created_at": "2026-02-08T14:30:12Z",
  "updated_at": "2026-02-08T14:30:12Z"
}
```

## List artifacts

```bash
curl "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact?page=1&page_size=20" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```json
{
  "data": [
    {
      "id": "artifact_2b4d6f8a1c3e5a7b",
      "project_id": "proj_7a5c3e1b9d2f4a6c",
      "name": "Q1 Support Summary",
      "type": "Document",
      "source": "User",
      "state": "Ready",
      "ingest_status": "Done",
      "has_content": true,
      "created_at": "2026-02-08T14:30:12Z",

  "has_previous_page": false,

```

## Get an artifact

```bash
curl "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact/artifact_2b4d6f8a1c3e5a7b" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```json
{
  "id": "artifact_2b4d6f8a1c3e5a7b",
  "project_id": "proj_7a5c3e1b9d2f4a6c",
  "name": "Q1 Support Summary",
  "type": "InlineText",
  "source": "User",
  "state": "Ready",
  "ingest_status": "Done",
  "has_content": true,
  "created_at": "2026-02-08T14:30:12Z",
  "updated_at": "2026-02-08T14:31:02Z"
}
```

## Update an artifact

```bash
curl -X PUT "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact/artifact_2b4d6f8a1c3e5a7b" \
  -H "Authorization: Bearer {api_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 Support Summary (final)",
    "description": "Finalized weekly summary notes.",
    "plain_text_content": "Ticket volume rose 12% week-over-week..."
  }'
```

Example response:

```json
{
  "id": "artifact_2b4d6f8a1c3e5a7b",
  "name": "Q1 Support Summary (final)",
  "description": "Finalized weekly summary notes.",
  "type": "InlineText",
  "source": "User",
  "ingest_status": "Done",
  "updated_at": "2026-02-08T14:45:21Z"
}
```

## Delete an artifact

```bash
curl -X DELETE "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact/artifact_2b4d6f8a1c3e5a7b" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```json
{
  "id": "artifact_2b4d6f8a1c3e5a7b",
  "state": "Deleted"
}
```

## Private artifact metadata

```bash
curl "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact/artifact_2b4d6f8a1c3e5a7b/private" \
  -H "Authorization: Bearer {api_token}"
```

Example response (schema is an untyped object):

```json
{}
```

## Download raw artifact content

```bash
curl "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact/artifact_2b4d6f8a1c3e5a7b/download" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```text
Ticket volume rose 12% week-over-week. Billing issues drove 38% of cases.
```

## Get parsed content

```bash
curl "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact/artifact_2b4d6f8a1c3e5a7b/parsed-content" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```text
Ticket volume rose 12% week-over-week.
Billing issues drove 38% of cases.
Priority bugs were resolved within 48 hours.
```

## Sync an artifact

```bash
curl -X POST "https://{base_url}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact/artifact_2b4d6f8a1c3e5a7b/sync" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```text
HTTP/1.1 202 Accepted
```

## Errors and edge cases

- **409**: artifact already processing.
- **413**: file or content too large.
- **404**: artifact not found.
