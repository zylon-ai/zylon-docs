## Overview
Developer Tools is the portal for developers using Zylon. Currently, it allows creating and managing API tokens.

This section is available to users with **Developer** permissions.

## Create API Token
To create an API token:

1. Click **Create API Token**.
2. Enter a name.
3. Generate the token.
4. Copy the provided token hash.

<Warning>
The token hash is only visible once. After closing the modal, it cannot be retrieved. Store the token securely.
</Warning>
