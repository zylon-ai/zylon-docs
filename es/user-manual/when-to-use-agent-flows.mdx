---
title: Cuándo usar agent flows
description: Usa agent flows para trabajo documental exhaustivo que va más allá de respuestas rápidas de chat.
icon: play
---

Usa **agent flows** en lugar de chat cuando necesites **procesamiento exhaustivo**:

- **Archivos grandes / cobertura completa del documento** (p. ej., RFPs de 300 páginas).
- Operaciones por lotes (haz muchas preguntas a la vez).
- Redacción de documentos más largos a partir de múltiples fuentes.
- Extracción estructurada a gran escala.

Los agent flows **se ejecutan en segundo plano**, por lo que no están optimizados para respuestas rápidas y parciales—**procesan todo lo necesario de la base de conocimiento** para producir resultados completos.

El resultado es un documento editable que se puede exportar como PDF o documento de Word para su posterior edición.
