El Backoffice de Zylon es el entorno de administración centralizado para gestionar la plataforma Zylon. Es independiente del Workspace y se usa para tareas de administración de plataforma y operaciones de desarrollo con API.

Las capacidades principales del Backoffice incluyen:

- Gestión de cuentas
- Gestión de roles y permisos
- Gestión de gateways
- Gestión de tokens de API
- Supervisión del estado de los servicios de la plataforma

El acceso al Backoffice requiere una cuenta válida con el rol adecuado:

- Rol **Operator** para administración de plataforma
- Rol **Developer** para flujos de tokens de API

Una misma cuenta puede tener varios roles. Las secciones visibles dependen de los roles asignados a esa cuenta.
