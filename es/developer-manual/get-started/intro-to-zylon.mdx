---
title: "Introducción a Zylon"
description: "Plataforma de IA autohospedada con endpoints de Messages compatibles con Anthropic y Workspace API."
---

Zylon es una plataforma de IA autohospedada que se ejecuta en tu infraestructura. Ofrece endpoints de Messages compatibles con Anthropic para interactuar con el modelo y una Workspace API para gestionar organizaciones y proyectos. En esta documentación usarás **dos APIs**:

- **API de ZylonGPT**: la capa de IA para respuestas del modelo, herramientas y chat con contenido.
- **Workspace API**: la API de administración para organizaciones, usuarios, proyectos e integraciones.
