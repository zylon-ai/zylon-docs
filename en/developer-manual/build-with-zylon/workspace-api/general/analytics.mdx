---
title: "Analytics"
description: "Track usage totals, rankings, and time series metrics."
---

Analytics endpoints summarize how your workspace is being used across projects and users.

## Basic request and response

Start with totals to get a usage snapshot from `GET /v1/app/organization/analytics/totals`.

```bash
curl "https://{base_url}/v1/app/organization/analytics/totals" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```json
{
  "projects_total": 12,
  "threads_total": 184,
  "interactions_total": 1420,
  "agent_flows_total": 36,
  "uploaded_artifacts_total": 87
}
```

## Active users ranking

```bash
curl "https://{base_url}/v1/app/organization/analytics/ranking/active-users?start_date=1738291200&end_date=1738972800&limit=5" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

## Time series

```bash
curl "https://{base_url}/v1/app/organization/analytics/timeseries?start_date=1738291200&end_date=1738972800&type=interactions" \
  -H "Authorization: Bearer {api_token}"
```

Example response:

```json
[
  { "date": "2026-02-02", "value": 18 },
  { "date": "2026-02-03", "value": 25 },
  { "date": "2026-02-04", "value": 21 }
]
```

## Errors and edge cases

- **400**: invalid date range or type.
- **403**: insufficient permissions.
