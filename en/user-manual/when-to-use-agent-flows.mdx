---
title: When to use AI Documents
description: Use AI Documents for comprehensive document work that goes beyond quick chat answers.
icon: play
---

Use **AI Documents** instead of chat when you need **comprehensive processing**:

- **Long files / full‑document coverage** (e.g., 300‑page RFPs).
- Batch operations (ask many questions at once).
- Drafting longer documents from multiple sources.
- Structured extraction at scale.

AI Documents **run in the background**, so they aren’t optimized for fast, partial answers—**they process everything that’s required from the knowledge base** to produce complete results.

The result is an editable document that can be exported as a PDF or a Word document for further edition.
