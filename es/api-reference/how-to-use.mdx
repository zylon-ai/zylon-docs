<Warning>
    Las API de Zylon están en desarrollo activo.
    Aunque no se anticipan cambios importantes que rompan la compatibilidad, pueden ocurrir ajustes menores.
</Warning>

Cuando se despliega en tu infraestructura en local, la plataforma Zylon desbloquea capacidades de IA exhaustivas a través de sus bloques de construcción listos para usar y pila tecnológica integrada:

![Resumen de arquitectura autocontenida de Zylon](/images/zylon-architecture.png)

La plataforma proporciona tanto API de bajo nivel (ZylonGPT) como de alto nivel (Workspace) que se ejecutan completamente dentro de tu infraestructura empresarial segura. Esto garantiza privacidad completa de datos, ya que ninguna información sale de tus instalaciones, permitiéndote implementar flujos de trabajo críticos con confianza.

Una ventaja clave de la plataforma Zylon es su modelo de uso ilimitado. A diferencia de los servicios de IA basados en la nube, no hay restricciones en tokens o ejecuciones de inferencia. Esto te permite integrar sistemas internos ilimitados y escalar el uso de tu plataforma de IA privada sin incurrir en costes adicionales.
