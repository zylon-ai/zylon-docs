La mayoría de aplicaciones con ZylonGPT siguen el mismo ciclo:

1. **Ingerir conocimiento** que quieres que el asistente use (Artefactos).
2. **Generar una respuesta** (Mensajes).
3. **Recuperar o analizar** ese conocimiento (Herramientas).

Esta sección se enfoca en las piezas que combinarás para construir funcionalidades reales.

## Mensajes

La API de mensajes es cómo hablas con el modelo: envías un historial de conversación y recibes el siguiente mensaje del asistente como bloques de contenido estructurados.

Empieza aquí: [API de mensajes](/es/developer-manual/build-with-zylon/working-with-messages)

## Artefactos

Los artefactos te permiten persistir contexto en Zylon (texto, archivos o URLs) dentro de una colección, para recuperarlo y reutilizarlo en futuras solicitudes.

Empieza aquí: [API de artefactos](/es/developer-manual/build-with-zylon/working-with-artifacts)

## Herramientas

Las herramientas permiten ejecutar operaciones específicas (como buscar artefactos, analizar datos tabulares o extraer una página web) como parte de una respuesta.

Empieza aquí: [Herramientas](/es/developer-manual/build-with-zylon/tools/introduction-to-tools)
