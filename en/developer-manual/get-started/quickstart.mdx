Start with ZylonGPT to send messages, or use the Workspace API to automate org and project setup.

## Prerequisites

- An API token. See [Token Management](/en/api-reference/token-management).
- Your Zylon hostname (replace `{BASE_URL}` in the examples).
- For Workspace API: an account and an organization created in the workspace.

## My first Zylon call

<Tabs>
  <Tab title="ZylonGPT API">
<Steps>
  <Step title="Send your first message">
    ```bash
    curl -X POST "https://{BASE_URL}/gpt/v1/messages" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "default",
        "max_tokens": 64,
        "messages": [
          { "role": "user", "content": "Write a one-sentence product description for Zylon." }
        ]
      }'
    ```

    Example response (your `id` and timestamps will differ):

    ```json
    {
      "id": "msg_d88cec8ffdf344dcafddb72026ead6be",
      "type": "message",
      "role": "assistant",
      "content": [
        {
          "type": "text",
          "start_timestamp": "2026-02-06T13:16:10.412540Z",
          "stop_timestamp": "2026-02-06T13:16:10.543884Z",
          "text": "Zylon is a private AI platform for building secure, tool-using assistants and workflows."
        }
      ],
      "model": "private-gpt",
      "stop_reason": "end_turn",
      "stop_sequence": null,
      "usage": {
        "input_tokens": 93,
        "output_tokens": 18
      }
    }
    ```
  </Step>

  <Step title="Stream responses (optional)">
    Set `stream: true` to receive **Server-Sent Events (SSE)**.

    ```bash
    curl -N -X POST "https://{BASE_URL}/gpt/v1/messages" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -H "Accept: text/event-stream" \
      -d '{
        "model": "default",
        "stream": true,
        "max_tokens": 32,
        "messages": [
          { "role": "user", "content": "Give a three-word tagline for a productivity app." }
        ]
      }'
    ```

    Example streamed events (truncated):

    ```text
    event: message_start
    data: {"type":"message_start","message":{"id":"msg_b55d29718361410bad4f80f8f67c6b72","type":"message","role":"assistant","content":[],"model":"private-gpt","usage":{}}}

    event: content_block_delta
    data: {"type":"content_block_delta","index":0,"block_id":"block_019c331ac11375cea03f7e0fb7a2a044","delta":{"type":"text_delta","text":"Focus"}}

    event: message_stop
    data: {"type":"message_stop"}
    ```
  </Step>
</Steps>
  </Tab>

  <Tab title="Workspace API">
<Steps>
  <Step title="Create a user linked to your account">
    ```bash
    curl -X POST "https://{BASE_URL}/v1/app/user" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "account_id": "acct_6c8a1f3b2d4e5f7a",
        "role": "AppAdmin",
        "user_info": {
          "email": "sasha@auroralabs.com",
          "name": "Sasha Patel",
          "given_name": "Sasha",
          "family_name": "Patel"
        }
      }'
    ```

    Example response:

    ```json
    {
      "id": "user_4b7c2a1d9e5f3c8b",
      "org_id": "org_2f3a9d1c7b5e4a8f",
      "account_id": "acct_6c8a1f3b2d4e5f7a",
      "name": "Sasha Patel",
      "email": "sasha@auroralabs.com",
      "state": "Active",
      "roles": ["AppAdmin"],
      "created_at": "2026-02-08T14:12:45Z"
    }
    ```
  </Step>

  <Step title="Authenticate">
    ```bash
    curl -X POST "https://{BASE_URL}/v1/auth/login" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "email": "sasha@auroralabs.com",
        "password": "S3cure!Passw0rd"
      }'
    ```

    Example response:

    ```json
    {
      "id": "acct_6c8a1f3b2d4e5f7a",
      "email": "sasha@auroralabs.com",
      "provider": "Credentials",
      "roles": ["AppAdmin"]
    }
    ```
  </Step>

  <Step title="Create a project">
    ```bash
    curl -X POST "https://{BASE_URL}/v1/app/project" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "name": "Customer Insights",
        "goal": "Summarize weekly support trends.",
        "deadline": 1709337600,
        "visibility": "Private"
      }'
    ```

    Example response:

    ```json
    {
      "id": "proj_7a5c3e1b9d2f4a6c",
      "name": "Customer Insights",
      "visibility": "Private"
    }
    ```
  </Step>

  <Step title="Upload an artifact">
    ```bash
    curl -X POST "https://{BASE_URL}/v1/app/project/proj_7a5c3e1b9d2f4a6c/artifact" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "type": "InlineText",
        "source": "User",
        "name": "Q1 Support Summary",
        "raw_content": "Ticket volume rose 12% week-over-week. Billing issues drove 38% of cases."
      }'
    ```

    Example response:

    ```json
    {
      "id": "artifact_2b4d6f8a1c3e5a7b",
      "project_id": "proj_7a5c3e1b9d2f4a6c",
      "name": "Q1 Support Summary",
      "type": "InlineText",
      "ingest_status": "Processing"
    }
    ```
  </Step>

  <Step title="Create a thread and query the artifact">
    ```bash
    curl -X POST "https://{BASE_URL}/v1/app/project/proj_7a5c3e1b9d2f4a6c/thread" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "name": "Weekly Support Brief",
        "visibility": "Private"
      }'
    ```

    ```bash
    curl -X POST "https://{BASE_URL}/v1/app/thread/thread_3f2a1c4d5b6e7f8a/interaction" \
      -H "Authorization: Bearer {API_TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "thread_id": "thread_3f2a1c4d5b6e7f8a",
        "artifact_ids": ["artifact_2b4d6f8a1c3e5a7b"],
        "pgpt": {
          "messages": [
            {
              "role": "user",
              "content": [
                { "type": "text", "text": "Summarize the artifact in three bullets." }
              ]
            }
          ],
          "max_tokens": 96,
          "temperature": 0.2
        }
      }'
    ```

    Example response:

    ```json
    {
      "id": "inter_5a7b9c1d3e5f7a9b",
      "state": "Completed"
    }
    ```
  </Step>
</Steps>

## Troubleshooting

- **401/403**: Your token is invalid or expired. See [Token Management](/en/api-reference/token-management).
- **404**: Check your base URL. For ZylonGPT, use `https://{BASE_URL}/gpt/v1/`.
- **400 (model not found)**: Use `"model": "default"` unless you’ve been given a specific model name.
- **405**: You’re using the wrong method. `/messages` is `POST`.
