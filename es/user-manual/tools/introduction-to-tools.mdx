---
title: Usar herramientas — resumen
description: Las herramientas amplían el agente de chat para razonamiento más profundo y análisis de datos.
---

El chat puede usar herramientas para mejorar la calidad y gestionar tareas de datos:

- **Pensamiento extendido** — permite al modelo pensar antes de responder; útil para problemas abiertos o de varios pasos.
- **Análisis numérico** — ejecuta consultas tipo SQL y gráficos contra un **único CSV** de tu base de conocimiento.
- **Obtener URL** — pega una **URL** web y el asistente obtendrá y usará el contenido de esa página para la respuesta.

> Puedes habilitar **múltiples herramientas juntas** (p. ej., Pensamiento extendido + Análisis numérico) cuando sea apropiado.
